Status / category label. Use `light` (soft tint) for most in-app states; `solid` for emphasis.

```jsx
<Badge color="success">Active</Badge>
<Badge color="danger" variant="solid">Overdue</Badge>
<Badge color="warning" dot>Pending</Badge>
<Badge color="primary" variant="light" pill>New</Badge>
```

Colors: primary/secondary/success/danger/warning/info/dark. `dot` adds a status dot; `pill` fully rounds.
