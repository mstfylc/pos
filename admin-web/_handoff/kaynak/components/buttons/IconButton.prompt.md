Square single-icon button — toolbars, table row actions, app header controls.

```jsx
<IconButton icon="setting-2" aria-label="Settings" />
<IconButton icon="trash" variant="light" color="danger" aria-label="Delete" />
<IconButton icon="more" variant="ghost" size="sm" aria-label="More" />
```

Defaults to `ghost`/`secondary`. Set `rounded` for a circular button. Shares the Button variant/color/size system.
